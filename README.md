# GI Joe API

A simple ASP.NET Core Web API built as a learning project to practice:

- C# fundamentals
- Object-Oriented Programming (OOP)
- Collections
- ASP.NET Core Web API basics
- Swagger / OpenAPI

This project exposes a basic API for managing GI Joe characters.

## How to Run the App

1. Navigate to the project folder:

   ```bash
   cd GiJoeApi

2. Run the application:

dotnet run

3. Open your browser and visit:

http://localhost:5116/swagger

## Tech Stack
- C#
- .NET  
- ASP.NET Core Web API
- SwaggerUI (Swashbuckle)

## Status

This project is under active development as part of a structured C# learning roadmap.
